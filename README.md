## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/basirulakhlakborno/anime-world-india-clone.git
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Set up environment variables:
   ```sh
   cp .env.example .env
   ```
   Edit `.env` and configure the following variables:
   
   ```env
   # API Base URL
   # API Repository: https://github.com/basirulakhlakborno/AniBiee-AnimeWorldIndia-scraper
   VITE_API_BASE_URL=http://localhost:3001/api
   ```
4. Start development server:
   ```sh
   npm run dev
   ```
